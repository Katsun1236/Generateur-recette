Tutoriel de Gestion du Marché

Cette page explique comment utiliser l'interface d'administration du marché pour ajouter, modifier et gérer les objets en vente.

Ajouter ou Modifier un objet

Le formulaire en haut de la page "Admin Marché" est utilisé pour créer et mettre à jour les objets.

  • Pour créer : Laisse le formulaire vide, remplis tous les champs, puis clique sur "Sauvegarder l'Objet".
  • Pour modifier : Clique sur le bouton "Modifier" d'un objet dans la liste. Le formulaire se remplira automatiquement. Fais tes changements, puis clique sur "Sauvegarder l'Objet".

Détails importants des champs :

  • Nom de l'objet : Ce nom doit être unique. C'est ce qui est affiché sur le site. Ex: Histerite Sword.
  • Prix d'achat : N'écris que le nombre, sans le "$" ni de points. Ex: 15000.
  • URL de l'image : C'est le chemin vers l'image. Le chemin part du dossier `js/` où se trouve le script. En général, ce sera ../images/market-icon/histeria-icon/nom-de-l-item.png.
  • Catégorie : Choisis une catégorie dans la liste pour que les filtres fonctionnent correctement.

Gérer les objets "À l'affiche"

Ce sont les objets mis en avant en haut de la page du marché. Tu peux en avoir 4 ou 5 pour un bel affichage.

  • Pour ajouter un objet à cette section, trouve-le dans la liste "Tous les objets" et clique sur le bouton "Mettre en affiche".
  • Pour le retirer, trouve-le dans la section "Objets à l'affiche" et clique sur "Retirer".

ATTENTION : La suppression d'un objet est une action définitive et irréversible. Sois certain de ton choix avant de cliquer sur "Supprimer".
