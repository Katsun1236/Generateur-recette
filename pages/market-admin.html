Gestion du Marché

Outils pour mettre à jour les articles du marché.

Comment ça marche ?

Le marché est géré par un fichier de configuration sur GitHub. Cette page vous aide à générer le code correct à copier-coller dans ce fichier.

  1. Remplissez le formulaire ci-dessous avec les informations de l'objet.
  2. Cliquez sur "Générer le code".
  3. Copiez le code généré dans le bloc qui apparaîtra.
  4. Ouvrez le fichier market-config.js sur GitHub, cliquez sur l'icône de crayon pour l'éditer, et collez le code à la fin de la liste `marketItems`.
  5. Validez les changements sur GitHub ("Commit changes").
